Email Sender API

A lightweight and ready-to-use JavaScript-based email sending utility built with Node.js, Express, Multer, and Nodemailer.

GitHub

Usage

  1. Include the script

    Add this to your HTML to use the email utility:

                                
                                <script src="https://jimboyz-send-mail.vercel.app/v1/email.js"></script>

  2. HTML Form Example

                                
                                <form onsubmit="sendEmail(); return false;">
                                    <input type="text" id="name" placeholder="Your Name" required>
                                    <input type="email" id="email" placeholder="Your Email" required>
                                    <input type="text" id="subject" placeholder="Subject" required>
                                    <textarea id="message" placeholder="Your Message" required>
                                    <input type="file" id="attachment">
                                    <button type="submit">Send Email</button>
                                </form>

  3. Send Email via JavaScript

    Using FormData

                                
                                function sendEmail() {
                                    const formData = new FormData();
                                    formData.append('name', `${document.getElementById('name').value} <${document.getElementById('email').value}>`);
                                    formData.append('subject', document.getElementById('subject').value);
                                    formData.append('message', document.getElementById('message').value);
                                    formData.append('recipient', 'mail@gmail.com'); //Replace with your email
                                    formData.append('attachment', document.getElementById('attachment').files[0]);
                                
                                    SMTP.send(formData)
                                    .then(response => {
                                        alert(response.message);
                                        clearInputFields(); // your custom function
                                    })
                                    .catch(error => {
                                        alert('Send failed. Please check the console.');
                                        console.error(error);
                                    });
                                }

    Using JSON Data as arguments 

                                
                                function sendEmail() {
                                    SMTP.send({
                                        name: document.getElementById('name').value,
                                        recipient: 'mail@gmail.com',
                                        subject: document.getElementById('subject').value,
                                        message: `Email: <${document.getElementById('email').value}>\n${document.getElementById('message').value}`,
                                    })
                                    .then(response => {
                                        alert(response.message);
                                        clearInputFields(); // To clear all the input fields.
                                    })
                                    .catch(error => {
                                        console.error(error);
                                    })
                                }

⚠️ Note:

This project is intended for learning and development purposes only.

Please be aware that using this tool in a public-facing website is not secure, especially if you hardcode your email credentials or recipients directly in the frontend.

Since the frontend JavaScript is accessible to users through browser developer tools (e.g., Inspect Element or View Page Source), your email address and other sensitive data can be exposed.

For production environments, always keep sensitive logic and credentials on the server side, and use authentication or secure API gateways to protect your email-sending functionality.
